# Code of Conduct

We want this community hub to stay welcoming and practical.

## Our standards

- **Be respectful.** Disagree with ideas, not people. No harassment, slurs, or pile-ons.
- **Stay on topic.** Bugs, features, recipes, and roasting craft belong here; spam and unrelated promotions do not.
- **Assume good intent** especially from newcomers and home roasters.
- **No secrets in public threads.** Do not post API keys, passwords, private URLs, or personal data you would not put on a postcard.

## Enforcement

Maintainers may edit, hide, lock, or remove content that violates this code, and may block repeat offenders. For sensitive conduct concerns, use the contact path described in **SECURITY.md** if you need a private channel.
